# GNU的哲学
## GNU是什么
GNU是一个自由软件操作系统，自由软件意味着使用者有运行、复制、发布、研究、修改和改进该软件的自由。GNU是一个100%由自由软件构成的操作系统。它由 Richard Stallman（rms）在1983年发起，并由许多珍视软件用户自由的人士共同协作开发。技术上，GNU大致类似Unix。但是和Unix不同，GNU给予其用户自由
