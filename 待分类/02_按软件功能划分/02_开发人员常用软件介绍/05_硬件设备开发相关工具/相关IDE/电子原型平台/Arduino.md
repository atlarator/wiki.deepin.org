# 简介

Arduino是一个软硬件平台，构建于开放原始码simpleI/O介面版，硬件部分是可以用来做电路连接的Arduino电路板，在软件平台IDE中编写程序代码，将程序上传到Arduino电路板并执行。

# 安装

`sudo apt-get install arduino`

# 卸载

`sudo apt-get remove arduino`
