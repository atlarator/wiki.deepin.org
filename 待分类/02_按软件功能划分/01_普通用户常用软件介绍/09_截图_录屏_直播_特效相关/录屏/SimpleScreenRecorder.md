## 简介

SimpleScreenRecorder是一款屏幕录制软件，它具有录制整个桌面、应用窗口、指定窗口大小以及音/视频同步，视频缩放等功能。

## 安装

`sudo apt-get install simplescreenrecorder`

## 卸载

`sudo apt-get remove simplescreenrecorder`
