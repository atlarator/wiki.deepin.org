## 简介

links是一个支持图形和文本模式的网页浏览器。

## 安装（Links仅支持文本模式版本）

```Shell
sudo apt-get install links
```

## 卸载

```Shell
sudo apt-get remove links
```

## 安装（Links支持图形和文本模式版本）

```Shell
sudo apt-get install links2
```

## 卸载

```Shell
sudo apt-get remove links2
```
