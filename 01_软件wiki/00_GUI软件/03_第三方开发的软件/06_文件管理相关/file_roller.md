## 简介

File Roller是GNOME桌面环境的归档管理器，它具有创建、修改、查看、提取等功能，以及可以保存为不同格式的文档。

## 安装

`sudo apt-get install file-roller`

## 卸载

`sudo apt-get remove file-roller`
