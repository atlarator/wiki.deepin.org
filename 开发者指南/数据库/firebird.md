## 简介

Firebird RDBMS 基于 InterBase 6.0 的代码开发 。

## 安装

`sudo apt-get install firebird3.0-server`	服务
`sudo apt-get install firebird3.0-utils`	组件
## 卸载

`sudo apt-get remove firebird3.0-server`
`sudo apt-get remove firebird3.0-utils`
